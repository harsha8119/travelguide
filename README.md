 Project Overview: 
This project is a **static travel destinations website** developed using **HTML and CSS**.  
The website provides information about popular international travel destinations, each presented on a separate, well-structured webpage.

The main goal of the project is to practice **frontend web development**, page navigation, and consistent UI design across multiple web pages.

Technologies Used
- **HTML5** – Page structure and content
- **CSS3** – Styling and layout

Key Learnings
- Building multi-page static websites
- Organizing HTML files efficiently
- Reusing CSS styles across pages
- Improving UI consistency and navigation flow
